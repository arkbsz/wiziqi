# 发布检查清单

每次准备发布新 APK 或更新联机服务前，按下面清单检查。

## 1. 代码与资源

- [ ] 前端代码修改已完成
- [ ] Android 资源已同步
- [ ] 默认联机地址已确认
- [ ] 无调试代码残留

## 2. 功能测试

- [ ] 本地对战可进入
- [ ] 联机对战可进入
- [ ] 创建房间正常
- [ ] 加入房间正常
- [ ] 落子同步正常
- [ ] 重开功能正常
- [ ] 返回主菜单正常

## 3. 文案与界面

- [ ] 中文无乱码
- [ ] 按钮文字正确
- [ ] 棋盘显示正常
- [ ] 手机屏幕适配正常

## 4. 服务端检查

- [ ] 云服务器服务在线
- [ ] `/health` 可访问
- [ ] 4000 端口已放行
- [ ] 公网地址可用

## 5. 版本管理

- [ ] 更新 `package.json` 版本号
- [ ] 更新 `android/app/build.gradle` 版本号
- [ ] 更新 `CHANGELOG.md`
- [ ] 确认 `.gitignore` 正常

## 6. 打包与发布

- [ ] 成功生成 APK
- [ ] 模拟器安装通过
- [ ] 真机安装通过
- [ ] 旧版本卸载/覆盖安装已验证

## 7. Git 操作

- [ ] 已提交代码
- [ ] 已推送到远程仓库
- [ ] 必要时已打 tag

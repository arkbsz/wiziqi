# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog.

## [1.1.0] - 2026-06-22

### Added

- Added project work tree document
- Added China cloud server deployment document
- Added pm2 ecosystem config for server deployment

### Changed

- Switched online play from older unstable flow to HTTP polling room sync
- Updated Android default online server to Huawei Cloud public server
- Simplified online screen so users no longer need to manually type server address
- Improved browser behavior to prefer local server when running on localhost

### Fixed

- Fixed Android compatibility and packaging issues for recent Android versions
- Fixed multiple menu and button interaction regressions
- Fixed online room creation and join flow integration
- Fixed various Chinese text display issues

## [1.0.0] - 2026-06-19

### Added

- Initial Gomoku mobile project
- Local play mode
- Android packaging baseline
